## console log vs code user snippets
> Open vs code editor
>> 1. Click File Menu
>> 2. Click preferences
>> 3. Configure user snippets
>> 4. Then search Javascript.json
>> 5. copy and paste the code
>> 6. {
	"Print to console": {
		"prefix": "log",
		"body": [
			"console.log($1);",
			"$2"
		],
		"description": "any text"
	}
}
